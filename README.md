# ir_web_api
InspectorRestaurant Express.Js API Server

## Getting Started
### Prerequisites
- node.js 8.x.x
- npm 3.x
- MongoDB 3.0

### Installing & Configuration
1) Install dependencies
```
npm install
```

### Run the server
```
npm start
```
